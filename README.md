# 8.1 x Git: Open Source Week

Welcome to **8.1 x Git** – a beginner-friendly open source initiative designed to help you begin your contribution journey with 8 thoughtfully crafted GitHub tasks.  
Sharpen your skills, one commit at a time. ✨

## Event Motto
**"Grow together—one pull request at a time!"**  
Whether you're new to open source or looking to enhance your GitHub skills, this is the perfect opportunity to learn, collaborate, and contribute meaningfully. 🌱

## What to Do?
This GitHub organization contains 8 repositories, each with a unique open source task.  
Explore each repository, understand the problem statement, and follow the instructions provided in the README of each. 📂

## Final Words
This is your chance to grow, learn, and collaborate in the open source world.  
We look forward to seeing your contributions. 🚀
